# Roundabout

## 프로젝트 소개
글을 읽고 정보를 추출하여 마인드맵을 그려 정답을 찾아내야 하는 퍼즐 게임입니다.  
정답을 찾아내기 위해 그린 마인드맵을 겹치면 특정 형태가 완성되는 것이 특징입니다.  
Unreal Engine 5.3을 기반으로 하고 있고 개인 프로젝트 형태로 진행하고 있습니다.
  
## 개발자
박지훈
